**Ask questions about your search performance in plain English — no dashboards, no exports, no waiting.**

Google Search Console holds some of your most valuable SEO data — impressions, clicks, rankings, and indexing status — but getting meaningful answers out of it usually means clicking through multiple reports, downloading CSVs, and stitching data together manually. This MCP connector brings all of that data directly into your AI client, so you can just ask what you want to know.

> _"Which pages lost the most impressions last month compared to the month before?"_
>
> _"Show me my top 20 queries by clicks for the last 90 days, filtered to mobile devices."_
>
> _"Which URLs have crawl errors or indexing issues right now?"_

## What You Can Ask

### Keyword & Query Performance

```
What are my top 50 queries by impressions over the last 28 days?
```

```
Which queries have a high impression count but a low click-through rate?
```

```
Show me queries where my average position improved by more than 5 spots this month.
```

### Page-Level Analysis

```
Which pages drove the most clicks last week?
```

```
Which pages have the highest impressions but lowest CTR — where am I leaving traffic on the table?
```

```
Show me all pages with an average position between 8 and 15 — these are my best ranking opportunities.
```

### CTR & Ranking Opportunities

```
What's my overall click-through rate trend over the past 6 months?
```

```
Which queries am I ranking in position 1–3 for but still getting low CTR?
```

```
Find queries where I rank on page 1 but have never gotten a click.
```

### Device & Country Breakdown

```
Compare my click and impression performance on mobile vs desktop for the last 30 days.
```

```
Which countries are driving the most organic traffic to my site?
```

```
Show me my top queries for tablet users specifically.
```

### Trend & Period Comparisons

```
Compare this month's search performance to the same period last year.
```

```
Did my impressions drop after the last Google algorithm update?
```

```
Show me week-over-week click trends for the past 3 months.
```

### Search Appearance

```
How many of my pages are appearing as rich results?
```

```
Which search appearance types are driving the most clicks — web results, image results, or video?
```
